## 2. win 添加小鹤双拼

注册标添加：

win + R 输入 `cmd`，输入：

```bash
reg add HKCU\Software\Microsoft\InputMethod\Settings\CHS /v UserDefinedDoublePinyinScheme0 /t REG_SZ /d "小鹤双拼*2*^*iuvdjhcwfg^xmlnpbksqszxkrltvyovt"
```
