# 插入链接

## 链接

```
[链接名称](网址 "可以不写的title，鼠标放在链接上会显示出来的文字")
```
[链接名称](网址 "可以不写的title，鼠标放在链接上会显示出来的文字")

<链接>

## 图片

```
![名称](链接 “title”)

或：

[名称][代号]

[代号]：链接 “title”
```

![名称](链接 "title")

## 引用和脚注

```
引用和脚注：
名词[^代号]

[^代号]：内容
在正文中[^代号]会自动编号成数字
```

代号[^1]

[^1]:内容
