# 锅具

> 杨光：<https://www.bilibili.com/video/BV1eC4y137Mj/>

## 1. 杨光 总结

### 1.1. 熟铁锅【买了】

随着火的调控，热得快，凉的快

30s 可以加热到 200 度。过高和过低都会粘锅

缺点：

-   热的不均匀，需要润锅和颠勺来解决
-   不要长时间接触酸性物质（醋、番茄），破坏油膜

### 1.2. 不粘锅【买了】

特氟龙 性能最好

只要不超过 250 度都还是安全的，正常不干烧是不会超过的

不能用铁铲，别炒排骨之类的有棱角的

消耗品，两三年换

锅烧糊了就不要用了

涂层脱落的话，吃了没事

### 1.3. 雪平锅【买了】

不锈钢导热慢，不均匀

### 1.4. 避雷

蜂窝式不粘锅 X

有凸起的铸铁锅

钛锅 X 很重
